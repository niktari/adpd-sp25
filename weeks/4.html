Description

This week continues our understanding of CSS! We will do a quick overview of web typography and how it corresponds to CSS properties. We’ll learn about in-browser developer tools that help build and debug our websites. We’ll then close with a deep dive into CSS layout.

  • Technical Exercise: CSS Layout Worksheet

    Complete the CSS Layout Worksheet. Make sure to save your own version of the worksheet in your CodePen account (click on the "Save" button or hit Command + S) to start editing it. Be sure to save your work along the way.

    This worksheet is part of your Attendance & Participation grade. To get full credit, you must hit all of the requirements below.

  • Accessible Book, Part 4: Refine Your Layout

    Using CSS, continue to flesh out the typography of your website. Consider the typographic system you’re developing: are you applying consistent styles to your headings, paragraphs, and any other levels of text? Start to think about your layout.

    Currently, you probably have a top-to-bottom, left-to-right reading experience. Do you want to keep that, or do something different? Consider CSS layout properties such as display, The Box Model (padding, border, margin), and positioning.
