Description

This week marks the end of our first unit, where we learned about the fluid nature of the web. To close out, we will hold formal crits for our first project, Accessible Book, and learn about more advanced CSS layout techniques like Flexbox. We’ll then introduce our second unit, which focuses on the interactive and generative potential of the web. To kick off our second project, we’ll talk about inputs and outputs as a starting point for ideation.

Assignments

  • Generative Tool, Part 1: Concept & Sketches

    Come up with three distinct ideas for a website that utilizes an input and an output. Determine what the input and output would be. For instance, when the user types something, will text appear? Consider who or what this tool is for. Do you want to make something that is useful, semi-useful or not useful at all? Bring references and sketches for each direction. Use any sketching tool of your choice (e.g. paper and pencil, Figma). References can be websites, artworks, music, photographs, or anything else that sparks an idea.

    This assignment is worth 5 out of 25 points of Project 2: Generative Tool. To get full credit, you must hit all of the requirements below.

  • Class Website: CSS & Responsiveness

    Take everything you’ve learned in HTML and CSS and update your class website! A reminder that your class website serves as an index that links out to all of your projects. Clean up your content and style your website any way you’d like. Make it responsive so it looks good when you view it on mobile. This class site is your own personal coding playground throughout the semester.

    This assignment is part of your Attendance & Participation grade. To get full credit, you must hit all of the requirements below.

  • Extra Credit: Flexbox Letterforms

    Explore the use of Flexbox to create gridded letterforms. You can choose a word, create an alphabet, or go crazy on one letter. If you want to lean into pixelation, great. If you want to create more organic letterforms, consider what other CSS properties you can use to break up the rigidity of the grid. Maybe border-radius? Softer colors? Shapes that are not just squares and circles? What kind of interaction, like :hover, can you add? How does it add meaning to the word or letter? Play around!

    Please be sure to study these CodePens for more reinforcement:

    1. Study Example: Responsive Flexbox Grid
    2. Study Example: Flexbox Word

    This exercise is worth 5 extra credit points towards Project 1. To get the full extra points, you must hit all of the requirements below.
