Description

We’re going to demonstrate everything we learned last week by doing a hackathon! To continue our focus on data-driven design, we’ll learn how to “normalize” our data so our values are easier to work with. We’ll then learn about JavaScript libraries and their advantages, specifically a drawing one called p5.js.
