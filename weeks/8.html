  • Tool, Part 1: Concept & Sketch

    Using Figma or any tool of your choice, sketch three distinct website ideas that incorporate user input and generate an output. Start by defining the core functionality of each concept. What is the primary interaction? For example, if a user types something, does text appear, transform, or generate something new?

    From there, consider additional features that could enhance the experience. Consider who or what this tool is for. Is it meant to be useful, semi-useful, or not useful at all? Exploring different directions will help refine your concept before development begins.
