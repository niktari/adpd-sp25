Description

This week, we will write our first line of code! We will learn about HTML, and how an HTML document turns into a website. We will also set up our codebase through GitHub and develop a workflow for managing our code throughout the semester.

  • Accessible Book, Part 2: Start with HTML

    Based on your site map, transition from paper to code by setting up your code environment and HTML files. Populate each of your HTML files with the respective content from your site map. Focus on content only (e.g. text, media). Do not worry about style right now. Do your best to develop a semantic structure that enhances accessibility and considers typographic hierarchy.
