  • Book, Part 2: Start with HTML

    Translate your site map into a working website by setting up your code environment and HTML files. Populate each file with its respective content based on your site map and content organization. Focus on structure, semantics, and hierarchy. This stage is only focused on content, so don’t worry about styling yet. Remember, no images! This is a type-only assignment.

    Consider the navigation as you're linking the pages together. For instance, can the reader only go forwards, or can they also go backwards in the narrative?
